# Inputs

The ways people give input that Apple documents under Inputs. Names are Apple's.
The descriptions are this project's own paraphrase.

Verify the latest at
https://developer.apple.com/design/human-interface-guidelines/inputs

Platform key. iOS, iPadOS, macOS, watchOS, tvOS, visionOS.

| Input | Platforms | What it is | When to use |
|---|---|---|---|
| Action button | iOS (Pro), Apple Watch Ultra | A configurable hardware button | Map it to one fast, frequent action |
| Apple Pencil and Scribble | iPadOS | Pencil input and handwriting to text | Drawing, marking up, and handwritten entry |
| Buttons (hardware) | iOS, iPadOS, watchOS | Physical side, volume, and crown buttons | Respect system behavior, do not override |
| Camera Control | iOS (iPhone 16 and later) | A press-and-swipe camera control | Camera capture and quick camera actions |
| Digital Crown | watchOS, visionOS | A rotating input | Scroll, adjust values, and precise input |
| Eyes | visionOS | Gaze as a targeting input | Pair look with pinch; design large comfortable targets |
| Focus and selection | tvOS, visionOS, keyboard | The focus model for non-touch input | Make the focused item obvious; design a clear focus order |
| Game controllers | iOS, iPadOS, macOS, tvOS, visionOS | Physical game controllers | Support standard layouts; show the correct glyphs |
| Gestures | iOS, iPadOS, visionOS | Touch and spatial gestures | Use standard gestures; avoid clashing with system gestures |
| Gyroscope and accelerometer | iOS, iPadOS, watchOS | Motion sensors | Use motion input sparingly and offer a non-motion alternative |
| Keyboards | all | Hardware and on-screen keyboards | Support shortcuts; pick the right on-screen keyboard type |
| Nearby Interaction | iOS | Ultra Wideband ranging and direction | Find and interact with nearby devices |
| Pointing devices | iPadOS, macOS, visionOS | Mouse, trackpad, and pointer | Support hover and precise pointer effects |
| Remotes | tvOS | The Siri Remote and other remotes | Design for clicks and swipes; keep focus movement simple |
| Siri | iOS, iPadOS, macOS, watchOS, tvOS | Voice assistant and intents | Expose key actions as App Intents and shortcuts |
| Spatial interactions | visionOS | Look, pinch, reach, and direct touch | Design comfortable targets and clear feedback in space |
