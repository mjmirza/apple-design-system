# Technologies

The Apple technologies and services you design for, documented under
Technologies. The last column notes whether Apple ships a downloadable design
template or glyph for it (see `10-resources-downloads.md` for the files). Names
are Apple's. The descriptions are this project's own paraphrase.

Verify the latest at
https://developer.apple.com/design/human-interface-guidelines/technologies

| Technology | What it is | When to use | Template or glyph |
|---|---|---|---|
| AirPlay | Streams audio and video to other devices | Offer AirPlay for media playback | Glyph |
| Always On | The dimmed always-on display | Keep key info legible while dimmed | no |
| App Clips | A small part of an app that runs without install | Offer a fast task from a link, NFC, or code | Template |
| Apple Pay | In-app and web payments | Offer fast, private checkout | Template |
| Apple Wallet | Passes, tickets, cards, and keys | Add passes and cards to Wallet | Template |
| AR (ARKit and RealityKit) | Virtual content placed in the real world | Place virtual content in the person's space | ARKit glyph |
| CarPlay | Apple's in-car interface | Offer driving-safe audio, maps, and comms | Framework templates |
| Game Center | Apple's gaming service | Add leaderboards, achievements, and multiplayer | Icon and glyph |
| Apple Intelligence | On-device and private-cloud generative features | Use system writing, image, and intent features responsibly | no |
| HealthKit | Health and fitness data | Read and write health data with consent | no |
| HomeKit | Home automation | Control accessories and scenes | Icon and glyph |
| iCloud | Sync and storage | Keep data in sync across a person's devices | no |
| ID Verifier and Tap to Pay | Read IDs and accept contactless payments on iPhone | Verify IDs or take payments without extra hardware | Tap to Pay template |
| In-App Purchase | Selling digital goods | Sell content and subscriptions per App Store rules | no |
| Live Activities | Live Lock Screen and Dynamic Island updates | Track an ongoing event live | Component and template |
| Live Photos | Motion photos | Capture and show Live Photos | no |
| Machine Learning | On-device models via Core ML | Run models privately on device | no |
| Maps | Apple Maps and MapKit | Show maps, places, and directions | no |
| Messages for Business | Chat with businesses in Messages | Offer support and commerce in Messages | no |
| NFC | Near-field reading and writing | Read tags and support contactless | no |
| Photo editing | Editing extensions for Photos | Offer edits inside the Photos app | no |
| ResearchKit | Apps for medical and health research | Build consented research studies | no |
| SharePlay | Shared experiences over FaceTime | Let people watch, listen, or do things together | no |
| ShazamKit | Audio recognition | Identify music and custom audio | no |
| Sign in with Apple | Private account creation and sign-in | Offer fast, private sign-in | Buttons template and logo |
| Siri and App Shortcuts | Voice and shortcut actions | Expose actions as intents and shortcuts | Template |
| TipKit | In-app tips | Teach features in context | Template |
| Widgets | Glanceable extensions | Show key info outside the app | Component |
